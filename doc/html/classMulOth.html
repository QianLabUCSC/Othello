Detailed Description

template<uint8_t L, typename keyType>
class MulOth< L, keyType >

Describes the data structure Grouped l-Othello. It classifies keys of keyType into 2^L classes.
The keys are first classifed into groups, each group is then maintained by one l-Othello.

Note
Query a key of keyType return L digit values ( valueIntType ). the types valueType and valueIntType are automatically generated.
