Detailed Description

template<uint8_t L, class keyType>
class Othello< L, keyType >

Describes the data structure l-Othello. It classifies keys of keyType into 2^L classes.

Note
Query a key of keyType return L digit values ( valueIntType ). the types valueType and valueIntType are automatically generated.
